Методы массиовов в Java Script

1:

push(): Добавляет элемент(ы) в конец массива.


let arr = [1, 2, 3]; arr.push(4); console.log(arr);

2:

pop(): Удаляет последний элемент массива и возвращает его значение.


let ark = [1, 2, 3]; let removedElement = arr.pop(); console.log(removedElement); console.log(ark);

3:

concat(): Возвращает новый массив, объединяя массивы, на котором он был вызван, с другими массивами и/или значениями.


let arr1 = [1, 2]; let arr2 = [3, 4]; let newArr = arr1.concat(arr2); console.log(newArr);

4:

forEach(): Вызывает функцию один раз для каждого элемента в массиве.


let art = [1, 2, 3]; art.forEach(function (element) { console.log(element); });

5:

map(): Создает новый массив с результатом вызова указанной функции для каждого элемента массива.


let arv = [1, 2, 3]; let newArv = arv.map(function (element) { return element * 2; }); console.log(newArv);

6:

filter(): Создает новый массив со всеми элементами, прошедшими проверку, задаваемую переданной функцией.


let ard = [1, 2, 3, 4, 5]; let filteredArd = ard.filter(function (element) { return element % 2 === 0; // Фильтруем только четные числа });

7:

every(): Проверяет, удовлетворяют ли все элементы массива условию, заданному в переданной функции.


let arr = [2, 4, 6, 8, 10]; let isAllEven = arr.every(function(element) { return element % 2 === 0; }); console.log(isAllEven); // true

8:

some(): Проверяет, удовлетворяет ли хотя бы один элемент массива условию, заданному в переданной функции.


let arr = [1, 3, 5, 7, 8]; let hasEvenNumber = arr.some(function(element) { return element % 2 === 0; }); console.log(hasEvenNumber); // true

9:

reverse(): Изменяет порядок элементов массива на обратный.


let arr = [1, 2, 3, 4, 5]; arr.reverse(); console.log(arr); // [5, 4, 3, 2, 1]

10:

sort(): Сортирует элементы массива на месте и возвращает отсортированный массив.


let arr = [3, 1, 4, 1, 5, 9, 2, 6]; arr.sort(); console.log(arr); // [1, 1, 2, 3, 4, 5, 6, 9]
